# 1522 문자열 교환

### solution
* b의 누적합 개수를 배열로 저장
* 슬라이딩 윈도우로 b의 개수만큼 이동시키면서 b의 개수가 몇개 있는 지 확인
* b의 전체 개수 - 현재 b의 개수 최소 값 갱신
