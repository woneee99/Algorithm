# 19598 최소 회의실 개수

### Solution
* 시작 지점을 기준으로 정렬 -> 두 회의실이 연달아 일어날 경우 확인 불가
	* TestCase일 경우 확인 불가
* 나의 풀이(boj19598)는 정렬 후, end를 모두 List에 넣고 확인했다.
* 이 방법보다는 블로그 참고 풀이(boj19598_1) 방법이 시간이 더 단축된다.
	* 시작 지점과 끝 지점을 모두 큐에 넣고 시작 지점일 때 회의실 개수를 증가하고, 끝 지점일 때 개수를 감소시킨다.
