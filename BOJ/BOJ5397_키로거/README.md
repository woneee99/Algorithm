# 5397 키로거

### Solution
* ListIterator<E> 인터페이스를 사용하면 양방향으로 탐색하면서 추가/삭제/검색이 가능하다.
* next(), previous() 메소드를 사용하면 리스트 내 커서 위치를 다음, 이전 방향으로 이동 가능하다.
* remove() 메소드는 next(), previous()로 반환된 가장 최근 데이터를 리스트에서 삭제한다.
* set(E e) 메소드는 next(), previous()로 반환된 가장 최근 데이터를 e로 대체한다.

### Classification
* 자료구조
* LinkedList & ListIterator
